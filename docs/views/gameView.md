# gameView.py


## view

Crée une classe qui prend en argument d’initialisation deux fonctions :
- Une fonction `init` : Une fonction qui sert à initialiser la vue. Elle doit attendre aucun argument et sera appelé à chaque fois que la vue va être affiché. Les variables déclarées dedans doivent être déclaré en début de fontion avec `global`.
- Une fonction `update` : Une fonction qui sert à gérer actualiser la vue. Elle doit attendre deux argument, la liste d'évènement et un objet surface qui correspond à l'écran.
