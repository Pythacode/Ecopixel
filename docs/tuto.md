# Tuto.py

## searchView

Une classe qui permet de gerer le tuto.

### __init__ :

Défini les étapes, et le texte qui correspond

### get_message :

Retourne le message qui correspond à l'étape

### get_advancement :

Retourne l'étape actuelle

### help :
        Affiche le message d'aide

### next :

Passe à l'étape d'après si et seulement si `source` est l'étape actuel
