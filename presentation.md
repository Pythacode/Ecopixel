Ce jeu a été créé dans le cadre des [trophées NSI](https://trophees-nsi.fr/) édition 2026. 
Ecopixel est un jeu dit "cozy" où le but est de planter le plus d'arbres possible. Pour obtenir des pousses d'arbre, il faut faire des recherches sur le moteur de recherche intégré dans le jeu. Les arbres font pousser des pommes, ces pommes peuvent être vendus pour acheter du matériel ou améliorer sa maison.
