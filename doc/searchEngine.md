# searchEngine.py

## search

Fonction qui permet de faire une recherche internet avec [Search1API](https://www.search1api.com/ "Search1API website").
Prends en argument :
- query : Une chaîne de caractère à rechercher
- max_result : Un entier, par défauts 10 qui indique le nombre maximum de résultat
- lang : Une chaîne de caractère qui indique la langue, par défaut le français

## searchView

Une vue pour la recherche
